## Instalar
Siga os passos abaixo!

```bash
> termux-setup-storage
(depois disso toque na permissão)
> pkg install git
> pkg install ffmpeg
> pkg install wget
> pkg install nodejs
> pkg install npm
> git clone https://github.com/BigBraim/BigBraim.git
> cd BigBraim
> bash install.sh
```

## Uso

